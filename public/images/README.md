# How to add images — 2 Oceans Global

Upload images INTO THIS FOLDER (public/images/) on GitHub.
The filename decides where the image appears. Formats: .jpg .png .webp

## Rules for a classy result
- Compress every image under ~300KB before uploading (tinypng.com)
- Keep one consistent style across all category shots — same background
  tone, same lighting. Consistency is what makes a catalog look premium.
- Wrong-shape images won't break anything (they auto-crop to fit),
  but shooting/exporting at the right shape frames best.

Upload the file → Vercel rebuilds → image is live in ~1 minute.
Delete the file → the styled placeholder returns automatically.
